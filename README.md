# vue2-framework
## 前端项目框架，是一个后端管理页面，包含用户管理、角色管理、菜单功能基础模块

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
